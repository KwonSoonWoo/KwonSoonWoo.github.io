**문제 상황**

다른 브랜치에서 작업해야할걸 master에서 작업함.

작업한걸 다른 브랜치에서 commit 및 push 해야하는 경우.

---



**방법**

1. **VCS->Git->Branches->Remote Branches->원하는 브랜치->CheckOut As..**
2. commit 및 push



위의 방법대로 하면 작업했던걸 그대로 원하는 브랜치에 push가 가능하다
